Christian Portfolio Website

How to use:
1. Open index.html in your browser.
2. Replace assets/img/christian.svg with your real photo.
   - If your photo is JPG, name it christian.jpg and change the image source in index.html.
3. Add your CV file here:
   assets/cv/christian-quema-cv.pdf
4. Replace these placeholders in index.html:
   - your.email@example.com
   - +63 900 000 0000
   - github.com/yourusername
   - facebook.com/yourpage
5. Replace project Live Demo and Source Code links with your real links.

Recommended folder:
portfolio/
├── index.html
└── assets/
    ├── img/
    │   └── christian.svg
    └── cv/
        └── christian-quema-cv.pdf
