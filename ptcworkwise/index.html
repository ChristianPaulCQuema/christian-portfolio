PTC WorkWise

A WorkWise Management System designed for Pateros Technological College, with role-based experiences for HR, Dean, Assistant Dean, and Employee users.

Project Overview

Role-based HR operations in one portal

PTC WorkWise supports account management, employee records, department management, document requests, report submissions, faculty schedules, tasks and reminders, announcements, and chat-style communication.

Role-based Modules

Separate tools for each user group

HR Module

Controls user accounts, approvals, employee records, departments, documents, reports, schedules, task calendars, announcements, and messaging.

Dean / Assistant Dean Module

Supports department oversight, professor management, document and report handling, curriculum tools, availability, schedules, tasks, chat, and profile settings.

Employee Module

Gives employees access to announcements, department details, document requests, report uploads, schedules, availability, tasks, chat, and profile management.
